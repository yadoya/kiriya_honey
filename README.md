# KiriyaHoney

odayaka? => odayakajanaiwane

## Installation

Add this line to your application's Gemfile:

    gem 'kiriya_honey'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install kiriya_honey

## Usage

	puts "Ichigo".odayaka? #=> 穏やかじゃないわね
	puts nil.odayaka? #=> この穏やかさ 穏やかじゃないわね。
